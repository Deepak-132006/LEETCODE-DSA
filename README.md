## 📂 Repository Structure

The repo is organized by **topic**, making it easy to find relevant patterns and practice by category.

```
LEETCODE-DSA/
├── ArrayPractice/       # Array manipulation & fundamentals
├── Backtracking/        # Combinatorial & recursive backtracking problems
├── HashMap/             # Hashing-based problems
├── Interview_Prep/      # Curated interview-focused questions
├── LeetCode/            # General LeetCode problem solutions
├── Leetcode-Badges/     # LeetCode achievement/badge tracking
├── Practice/            # General practice problems
├── PrefixSum/           # Prefix sum technique problems
├── Recursion/           # Recursive problem-solving
├── Rotation/            # Array/matrix rotation problems
├── Searching/           # Binary search & search algorithms
├── SlidingWindow/       # Sliding window technique
├── Sorting/             # Sorting algorithms & applications
├── String/              # String manipulation problems
└── TwoPointers/         # Two-pointer technique problems
```

Each folder contains standalone `.java` files, with one file typically corresponding to one LeetCode problem.

## 🚀 Getting Started

### Prerequisites
- [Java JDK 8+](https://www.oracle.com/java/technologies/downloads/) installed
- (Optional) [VS Code](https://code.visualstudio.com/) with the Java Extension Pack

### Running a Solution

```bash
# Clone the repository
git clone https://github.com/Deepak-132006/LEETCODE-DSA.git
cd LEETCODE-DSA

# Navigate to any topic folder
cd Sorting

# Compile and run a solution
javac SolutionFileName.java
java SolutionFileName
```
